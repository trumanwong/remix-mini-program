## 安装指引

本项目使用Vue3+NutUI编写，[后端地址](https://github.com/trumanwong/remix-api)


```shell
# 1. 修改project.config.json的app_id为你的小程序app_id
$ npm install
# 开发环境
$ npm run dev:weapp
# 正式环境
$ npm run build:weapp
```
